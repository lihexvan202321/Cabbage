# 模拟器模拟器 - 更新说明 v1.1

## 新增功能

### 1. 成本价系统
- 每个模拟器的成本价为 **¥1.5**
- 在左侧状态面板显示成本价
- 销售结果中会显示成本和利润
- 帮助你更好地制定定价策略

### 2. 价格调整
- 正常价格从 ¥5 调整为 **¥3**
- 价格滑块范围扩大到 2-20 元
- 超过15元会有警告提示

### 3. 每日多次销售
- 每天可以进行 **2次销售**
- 第1次销售完成后需要等待 **10秒**
- 等待期间可以调整价格
- 等待结束后可以进行第2次销售

### 4. 销售倒计时
- 销售完成后显示10秒倒计时
- 倒计时期间按钮禁用,显示剩余时间
- 倒计时结束后按钮恢复,可以进行第2次销售

### 5. 价格违规机制 ⚠️
- **单价超过15元会被市场监督管理局带走!**
- 价格超过15元时,价格显示会变红并显示警告标志
- 如果尝试以超过15元的价格销售,游戏立即结束
- 游戏结束画面会显示你被带走的原因和最终经营数据

## 改进内容

- 在定价策略处显示成本价参考

### 游戏体验
- 更真实的经济系统(考虑成本)
- 更有策略性的定价(需要在利润和合法性之间平衡)
- 每日两次销售提供更多操作机会
- 等待期间可以调价增加了灵活性

## 游戏技巧

1. **合理定价**: 建议价格在 ¥3-5 之间,既能保证利润又不会吓跑客户
2. **成本控制**: 记住每个模拟器成本是 ¥1.5,定价低于这个值会亏本
3. **两次销售**: 充分利用每日两次销售机会,可以在第一次销售后根据情况调整价格
4. **等待期调价**: 10秒等待时间是调整价格的好时机,可以根据第一次销售情况优化第二次定价
5. **不要贪心**: 价格超过15元会直接游戏结束,切勿哄抬物价!

## 技术细节

### 销量计算公式更新
```javascript
价格系数:
  价格 ≤ 2:  2.0  (超低价)
  价格 2-4:  1.5  (低价)
  价格 = 3:  1.0  (正常价格)
  价格 4-7:  0.8  (偏高)
  价格 7-10: 0.5  (高价)
  价格 10-15: 0.3 (超高价)
  价格 > 15: 0.1  (天价 - 会被抓!)
```

### 利润计算
```javascript
利润 = 收入 - 成本
收入 = 销量 × 实际价格
成本 = 销量 × 1.5
```

### 销售流程
1. 点击"开始第1次销售"
2. 计算销量并扣除库存
3. 显示销售结果(含成本和利润)
4. 启动10秒倒计时
5. 等待期间可以调整价格
6. 倒计时结束后点击"开始第2次销售"
7. 两次销售完成后点击"下一天"
